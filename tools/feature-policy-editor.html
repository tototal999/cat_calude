ClaudeCat 功能政策管理

勾選要開放的功能，儲存 feature-policy.json 後由管理者重新打包 EXE。

政策烤在 EXE 裡，改完必須重新打包才會生效。
這樣使用者沒有可以編輯或刪除的政策檔（放在 exe 旁的檔案，刪掉就等於全部解鎖）。 但這仍是部署控制、不是資安機制：會解開 PyInstaller 封裝的人依然拿得到。 重點是「用記事本改一行」這條路被堵住了。

① 現有政策

使用 open-feature-policy-editor.bat 啟動時會自動載入專案根目錄的 feature-policy.json；直接單獨開啟 HTML 時才需要手動選檔。 公司 LLM 端點設定在另一個檔案（company-defaults.json），與這裡無關、不會被影響。
